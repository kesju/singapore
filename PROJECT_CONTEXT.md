# Project Context

## Purpose

This repository organizes planning materials for a trip to Singapore.

## Current Scope

- Itinerary planning
- Booking storage
- Places to visit
- Budget tracking
- Packing checklist

## Repository Structure

| Path | Purpose |
| --- | --- |
| `README.md` | Short project introduction |
| `itinerary.md` | Daily trip plan |
| `bookings/` | Booking confirmations and related documents |
| `places.md` | Food, attractions, shopping, and notes |
| `budget.md` | Estimated and actual trip costs |
| `packing.md` | Packing checklist |

## Working Notes

- Keep trip details in Markdown unless a booking or document requires another format.
- Store reservation files, screenshots, PDFs, and confirmations in `bookings/`.
- Avoid committing sensitive personal information unless this repository is private and intended for that use.
